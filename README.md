## Kurulum

1. `git clone https://github.com/kullaniciadi/learn-data-science-basics.git`
2. `cd learn-data-science-basics`
3. `conda create --name ds_env python=3.9`
4. `conda activate ds_env`
5. `pip install -r requirements.txt`
